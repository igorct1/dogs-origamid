# Deploy

[Deploy - Netlify](https://igorct-dogs-origamid.netlify.app/)

## Como utilizar?

```javascript
- Clone o repositório
$ git clone git@github.com:igorct1/dogs-origamid.git

- Instale as dependências
$ npm install

- Rode o Projeto
$ npm run dev

```
